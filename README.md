# 冰鉴
#### 用到的Animate动画效果 

animate__fadeInDown  从上而下
animate__fadeInDownBig 从上而下 大距离
animate__fadeInUp  从下而上
animate__fadeInUpBig  从下而上 大距离

animate__slideInUp  完美运动

animate__fadeOut 延迟出现
